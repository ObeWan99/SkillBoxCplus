# Cmake

1) создаём 

```sh

    nano CMakeLists.txt
```

2) затем создаём директорию 

```sh

    mkdir build
```

3) переходим 

```sh

    cd build
```

4) формируем файлы сборки 

```sh

    cmake ..
```

5) собираем проект 

```sh

    make
```
6) запускаемый бинарый файл main.out

```sh

    ./home/obewan/skillBox/build/src/operation.out
```
